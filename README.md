## 프로그램 명
Zero to Agent - 첫 AI 에이전트 만들기

## 목적
    1. 재직자의 AI 도구 활용 역량을 강화하고 실무 적용 가능성을 확인한다
    2. 비개발 직군(마케터·기획자·디자이너)이 직접 제작한 AI 에이전트 사례를 공유한다
    3. AI 에이전트 제작 과정에서 얻은 실전 노하우와 인사이트를 나눈다
    4. 다양한 직군 간 AI 활용 경험을 교류하며 업무 적용 아이디어를 발굴한다
    5. 재직자의 디지털 전환 역량 개발을 위한 지속적 학습 동기를 부여한다
